Pirate's Cove - Table-top board game

  • Designed this social gesture-based board game as part of an assignment.
  • Initially brainstormed, gathering inspirations from games like Citadels and Cash_n_Guns.
  • Designed the core actions - Attack, Defend and Loot with appropriate intuitive gestures for each.
  • Based on these actions, I designed the game around the Pirates theme and added a dice mechanic.
  • Playtested and found out that it was fun to play as pirate characters, especially with the gestures.
  • Through playtesting, also found that gameplay can be made more engaging and fun by adding special action cards.
  • Hence added 9 special action cards, that alter the gameplay in fun ways.
  • Game can be played by 3 - 6 players.
