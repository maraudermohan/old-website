Twitmania - Twitter-based iOS game

  • This personal project is about a casual twitter-based quiz game that I co-developed with my cousin, who worked on the implementation and development.
  • We designed the game to be quick and casual with challenging achievements.
  • Basic gameplay is guessing the user who tweeted the current tweet.
  • Scores change based on normal tweets, re-tweets and verified tweets.
  • Future versions will include lifelines that can be purchased and upgraded from the store - a strategic element in the gameplay.
  • This game acted as our springboard into mobile game development.
  • The biggest takeaway from this project - experience with marketing and monetization.
