Work Summary

Project Promo

'Interactive powerpoint for teachers!'

  • Designed a interactive visual teaching tool for teachers.
  • Designed to be used for any subject, as the content is completely customizable.
  • Teachers can easily upload any set of images, texts and hyperlink (both video and website) - using a google document.
  • Assets are imported into the tool automatically from the doc.
  • Imported assets can be manipulated however necessary, using the Wand.
  • Thus, the tool can be customized to teach a concept, conduct a quiz or even present a topic interactively.

'Elapsed time in Space!'

  • Designed a clock game for elementary school students to teach the concept of Elapsed time.
  • Players control the 'minute' hand of the clock using the wand and the 'hour' hand updates automatically.
  • Designed the game to cater the syllabus of different grades as it is completely customizable.
  • Teacher can easily upload the questions/quiz - using a google document.
  • Modes change based on values of three variables (entered by the teacher): Start time, elapsed time and end time.
  • Built the multiplayer version for this game, by displaying two clocks, each controlled by a different wand.

'Scrabble meets Ice Age!'

  • Designed a reading game for elementary school students (game mechanic is similar to Scrabble).
  • Players play two modes - first encoding mode to build words using letters and second decoding mode to identify similar sounding words.
  • Teacher enters the letters and word patterns based on their own curriculum - using a google document.
  • Multiple wands can be used at the same time to play multiplayer version.

Educational games & tools for K-12 students

  • Project was a collaboration between our Carnegie Mellon student team and Elizabeth Forward School District (EFSD), Pittsburgh.
  • Designed fun, interactive, educational games using SMALLab platform.
  • SMALLab is an embodied learning environment with motion-capture technology that tracks students' 3D movements using a 'wand'.
  • Game is projected on the floor using a hi-res projector and users interact with the video using the sand (tracked by the motion-capture cameras).
  • Goal was to reinforce curricular concepts through fun and engaging games, with teachers being the facilitators.
  • As Lead designer, I worked on balancing curriculum with simplistic 'game-experience' and made sure all games were highly customizable.
  • Designed three interactive experiences :
    a) Teaching tool
    b) Clock game
    c) Reading Game
