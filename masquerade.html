Masquerade Murder - Sifteo cubes board game

  • A personal game design project using Sifteo cubes as the platform.
  • Since Sifteo cubes are no longer supported, this game-design can easily be translated to current NFC/Bluetooth enabled, mobile devices.
  • 'Masquerade Murder' is an unique social - strategy board game, for 2-5 players.
  • The players take up the roles of the five main characters, each of whom has his/her own mission and an unique ability necessary to succeed.
  • Designed this to be a local multiplayer game using multiple devices. Each player having their own device and communicate with each other via NFC/Bluetooth.
  • Built a paper prototype and successfully playtested the game.
